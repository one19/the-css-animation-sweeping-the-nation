# The CSS Animation Solution

Pure CSS solution using CSS variables:

```css
animation-delay: calc(
  (
    /* calculate distance from final cell to THIS cell */
    max(abs(--x - --success-start-x)) +
    max(abs(--y - --success-start-y))
  ) * 0.2s
);
```
