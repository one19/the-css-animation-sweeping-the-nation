# The CSS Animation Solution

<Column>

Pure CSS solution using CSS variables:

```css
/* if the success vars are set, run the animation! */
#puzzle[style*='--success-start-x'][style*='--success-start-y'] & {

  /* very simple 1 keyframe animation */
  animation: ${successWave} 1s linear forwards;

  /* delay * distance */
  animation-delay: calc(
    (
      /* manhattan distance */
      max(abs(attr(data-x) - --success-start-x)) +
      max(abs(attr(data-y) - --success-start-y))
    ) * 0.2s
  );
}
```
